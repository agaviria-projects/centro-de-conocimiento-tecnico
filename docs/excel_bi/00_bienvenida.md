# 🏢 ¿Por qué aprender Excel BI?

En muchas empresas los usuarios trabajan principalmente con Excel.

Aunque Power BI es una herramienta muy potente, no siempre está implementada o disponible para todos.

Power Pivot permite construir modelos de análisis avanzados sin salir de Excel, aprovechando herramientas que ya forman parte del entorno de trabajo de la organización.

Por esta razón, dominar Excel BI representa una habilidad muy valiosa para un Analista de Datos.

# 🔄 Flujo de trabajo

El flujo recomendado para un proyecto de análisis de datos es:

```text
Archivos Excel
      │
      ▼
Power Query
(Limpieza de datos)
      │
      ▼
Modelo de Datos
(Relaciones)
      │
      ▼
Power Pivot
(Medidas DAX)
      │
      ▼
Tabla Dinámica
      │
      ▼
Dashboard
```

Cada etapa cumple una función diferente.

Durante este curso aprenderemos cuándo utilizar cada una.
